# 🧾 Billing Software (Python Tkinter)

A **Desktop Billing Application** built using **Python and Tkinter**.
This project allows users to generate retail bills for different categories of products such as **Medical Items, Grocery Items, and Cold Drinks**. The software calculates totals, taxes, and generates a printable bill that can also be saved and searched later.

---

## 🚀 Features

* 🧑 Customer details input (Name, Phone Number)
* 🛒 Product categories:

  * Medical Products
  * Grocery Items
  * Cold Drinks
* 🧮 Automatic price and tax calculation
* 🧾 Bill generation system
* 💾 Save bills as text files
* 🔍 Search previously saved bills
* 🧹 Clear all fields functionality
* ❌ Exit application safely

---

## ⚙️ How It Works

1. Enter **Customer Name and Phone Number**
2. Input the **quantity of products**
3. Click **Total** to calculate prices and taxes
4. Click **Generate Bill** to create the bill
5. Optionally **Save the bill** for future reference
6. Use **Search Bill** to retrieve previous bills

## ▶️ How to Run the Project

1. Install Python (3.x recommended)

2. Clone the repository


3. Navigate to the project folder
